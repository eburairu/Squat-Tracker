# データのエクスポート・インポート機能

## 概要
ユーザーがローカルストレージに保存されたデータをJSONファイルとしてエクスポート（バックアップ）し、
またそのファイルをインポート（復元）できる機能を追加する。

## 目的
- ブラウザのキャッシュクリアや端末変更によるデータ消失を防ぐ。
- ユーザーにデータの所有感と安心感を提供する。

## 対象データ
キーが `squat-tracker-` で始まるすべてのローカルストレージアイテム。
- 履歴 (`squat-tracker-history-v1`)
- 設定 (`squat-tracker-workout-settings`, `squat-tracker-voice`, `squat-tracker-theme`)
- プリセット (`squat-tracker-presets`)
- ボスバトル進行状況 (`squat-tracker-boss-v1`)

## UI変更
- `index.html`: 「データ管理」カードを追加（エクスポートボタン、インポートボタン）。
- `app.js`: `DataManager` オブジェクトを実装し、処理をカプセル化。

## 注意点
- インポート時は既存データを上書きするため、警告を表示する。
- インポート完了後はページをリロードして反映させる。

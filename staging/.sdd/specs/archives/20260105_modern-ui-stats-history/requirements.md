# 要件定義書

## 機能概要

ネイティブアプリのようなモダンなUIに刷新し、統計値の表示と過去実績の自動保存を追加する。

## ユーザーストーリー

- ユーザーとして、使いやすく洗練された画面でスクワットを記録したい。なぜなら集中してトレーニングしたいから。
- ユーザーとして、今日の進捗や累計の統計を確認したい。なぜなら成果を把握したいから。
- ユーザーとして、過去の実績が自動で残ってほしい。なぜなら継続の記録を後から見返したいから。

## 機能要件

### 要件1：モダンUI

- アプリ全体のレイアウト、タイポグラフィ、配色を刷新してネイティブアプリ風の見た目にする。
- 受入基準：
    - [ ] 主要画面のUIが刷新され、旧デザインが残っていないこと
    - [ ] モバイルとデスクトップで表示崩れがないこと

### 要件2：統計値の表示

- セッション内の進捗と、保存済み実績から導出される統計値を画面に表示する。
- 受入基準：
    - [ ] セット数・回数などの進捗がリアルタイムに見えること
    - [ ] 累計回数や直近の実績などの統計が表示されること

### 要件3：過去実績の自動保存

- トレーニング完了時に実績を自動で保存し、次回起動時に参照できるようにする。
- 受入基準：
    - [ ] 完了時に実績が自動保存されること
    - [ ] ページ再読み込み後も過去実績が確認できること

## 非機能要件

- パフォーマンス：統計計算と保存処理が体感的に遅延しないこと
- セキュリティ：機密情報を保存しないこと

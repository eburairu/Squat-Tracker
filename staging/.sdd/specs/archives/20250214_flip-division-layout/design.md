# 設計書

## 方針

- タイマー領域内の割り算クイズをDOM上で上部に配置し、視覚的な優先度を上げる。
- 割り算クイズ領域に上下反転のスタイルを付与し、画面上部から見る利用者が読みやすい向きにする。

## 変更点

- `index.html`
  - `.timer` 内の `.quiz-area` をタイマー数値/ヒントより前に移動する。
- `styles.css`
  - `.quiz-area` に上下反転のための `transform` を追加する。
  - 余白調整のために上下マージンを見直す。

## 影響範囲

- UIレイアウトのみ。ロジックやデータ更新には影響しない。

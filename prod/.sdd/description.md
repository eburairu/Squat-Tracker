# Current Task: Adventure Mode Implementation

## 概要
ユーザーのモチベーション維持と達成感向上のため、RPG要素を強化した「アドベンチャーモード（ワールドマップ機能）」を実装する。

## 主要機能
1.  **ワールドマップ進行 (すごろく形式)**
    *   敵を倒すことでコマが進む、すごろく型の進行システム。
    *   複数のエリア（草原、森、洞窟など）で構成される。
2.  **背景統合UI**
    *   トレーニング画面の背景に現在地のマップ風景を統合。
    *   常に自分の進行状況が視覚的にわかるようにする。
3.  **王道ファンタジー世界観**
    *   RPGらしい没入感を提供する。

# 設計

## 方針

- AGENTS.md を「基本方針」「テスト手順」「プロジェクトコマンド」「共有スキル」の4セクションに整理する。
- 手順は番号付きリストに統一し、必要な前提や注意点を短く明記する。

## 構成案

1. 基本方針
   - 回答言語（日本語）
   - SDDフロー遵守
2. テスト手順（Playwright）
   - `npm test` の必須実行
   - 部分テスト実行時の `npm run test:screenshot`
3. プロジェクトコマンド
   - Install/Test/Lint
4. 共有スキルカタログ
   - 参照手順と利用時の出力形式
   - スキル一覧

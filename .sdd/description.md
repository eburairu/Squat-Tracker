# システム概要

Squat-Tracker は、日々のスクワットトレーニングをサポートし、継続を促進するためのWebアプリケーションです。
リズムガイド、ゲーミフィケーション（ボスバトル、実績バッジ）、データ可視化（ヒートマップ）を組み合わせ、ユーザーのモチベーション維持を図ります。

## 主要機能

### 1. トレーニング・コア
- **リズムガイド**: 3段階（Down, Hold, Up）のフェーズ管理とタイマー。
- **音声読み上げ**: Web Speech API を使用したカウントと応援。
- **センサー検知**: Device Orientation API を使用したスクワット深度判定。

### 2. ゲーミフィケーション
- **ボスバトル (RPGモード)**:
    - スクワット回数に応じたダメージ計算。
    - 放置によるボスHP回復（継続促進）。
    - 討伐数カウントとモンスターの変化。
- **実績バッジシステム**:
    - 特定条件（累計回数、連続日数、討伐数など）達成によるバッジ付与。
    - バッジ獲得時の演出とコレクション表示。
- **レベルアップ**: 累計回数に基づくユーザーランク/レベル。

### 3. データ管理と可視化
- **活動ヒートマップ**: 過去1年間の活動状況をカレンダー形式で可視化。
- **履歴管理**: ローカルストレージへのセッション保存。
- **データ永続化**: JSON形式でのエクスポート/インポート機能。
- **プリセット (マイ・メニュー)**: トレーニング設定の保存と呼び出し。

## アーキテクチャ

### フロントエンド
- **HTML/CSS/JS**: フレームワークなしのVanilla JS構成。
- **Single Page Application (SPA)**: `index.html` 1ファイルに主要UIを集約（タブ切り替え等はCSS/JSで制御）。

### データストア
- **localStorage**:
    - `squat-tracker-history`: トレーニング履歴
    - `squat-tracker-achievements`: 獲得バッジ
    - `squat-tracker-boss-v1`: ボスバトルの状態
    - `squat-tracker-presets`: 設定プリセット
    - その他設定値

### デプロイ
- **GitHub Pages**: `main` (prod) / `develop` (staging) ブランチへのプッシュで自動デプロイ。

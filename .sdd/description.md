# 機能開発の概要

## デイリーミッション機能の実装

### 目的
ユーザーに「毎日アプリを開く理由」と「明確な短期目標」を提供し、継続率を向上させる。

### 機能概要
- **日替わりミッション**: 毎日0時に3つのミッションを更新して提示する。
- **ミッション内容**:
    - 継続系: アプリ起動、連続ログインなど
    - 努力系: 指定回数のスクワット、セット数達成など
    - （センサーモード関連は除外）
- **報酬システム**:
    - ミッション達成時に報酬として**武器を100%の確率で獲得**できる。
    - 既存の武器ドロップロジックを活用・拡張する。
- **UI**:
    - トップ画面（ボスカードの上部）に「今日のミッション」カードを配置。
    - 達成状況と報酬受取を視覚的に分かりやすく表示。

### 期待される効果
- ログイン率の向上。
- 武器収集の楽しみによるモチベーション維持。
- 「今日は何をしよう？」という迷いの解消。

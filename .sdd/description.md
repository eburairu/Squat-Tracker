# System Overview: Squat-Tracker

## 概要
Squat-Trackerは、スクワットトレーニングをRPG要素やゲーミフィケーションで楽しく継続させるためのWebアプリケーションです。

## 最近の実装機能 (2026-01-25)
### 称号システム (Title System)
- **称号獲得**: 実績解除の報酬として、「Prefix（形容詞）」と「Suffix（名詞）」を獲得。
- **カスタマイズ**: 獲得したパーツを自由に組み合わせて、ユーザー独自の称号を設定可能。

### アドベンチャーモード (Adventure Mode)
- **ワールドマップ進行**: 敵を倒すことでコマが進む、すごろく型の進行システム。
- **エリア攻略**: 草原、森、洞窟など複数のエリアで構成され、各エリアのボスを倒して次へ進む。
- **背景統合**: トレーニング画面の背景が現在地のエリアテーマに合わせて変化。

### その他主要機能
- **ボスバトル**: スクワット回数に応じたダメージ計算とモンスター討伐。
- **装備・インベントリ**: 武器の収集・強化システム。
- **デイリーミッション**: 毎日更新される課題と報酬。
- **計算クイズ**: トレーニング中の脳トレ要素（協力/お邪魔モード）。
